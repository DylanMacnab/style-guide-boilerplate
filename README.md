# Style Guide Boilerplate

<p>View the living <a href="http://dylanmacab.github.io/style-guide-boilerplate/">Style Guide Boilerplate</a></p>

<h2>Getting Started</h2>
<p>Living style guides improve consistency, coding standards, and communication among design and development teams. I created this Style Guide Boilerplate as a way speed up the process of creating new style guides. With each example I inlude reusable classes, code examples, and tips for how and when to use a style.</p>
